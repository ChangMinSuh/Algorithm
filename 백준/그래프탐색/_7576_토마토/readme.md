# 토마토

- https://www.acmicpc.net/problem/7576

## 해설
bfs

- 2차원 배열 arr에 정보 저장.
    - 1 이면 토마토 익음. visited 대체 가능.
- 날짜를 구해야 하니 n차별 개수만큼 큐를 실행.
    - 1일
        - arr[y][x] 값이 1인 토마토 만큼 큐에 삽입
        - 당시 큐에 들어있는 만큼만 for문을 사용해 인접한 안 익은 토마토(0) 큐에 삽입.(이때 1로 바꿔줌)
    - n일
        - 당시 큐에 들어있는 만큼만 for문을 사용해 인접한 안 익은 토마토(0) 큐에 삽입.(이때 1로 바꿔줌)
    - 큐가 비워질때 까지 반복
- 위 작업을 마친 후
    - 안익은 토마토(0)가 있다면 -1
    - 그 외에는 날짜만큼.
