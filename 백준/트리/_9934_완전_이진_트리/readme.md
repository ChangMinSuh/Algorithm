# 완전 이진 트리

- https://www.acmicpc.net/problem/9934

## 해설

bfs

- 중위 순회
  - 특징으로 출력한 순서가 이진 트리의 왼쪽에서 오른쪽 순이다.
- 이진트리이며,그중에서도 **포화 이진트리.**
  - 노드의 개수가 (2^N - 1) 이기 떄문.
- 배열의 가운데 값이 나머지 노드들의 조상노드!
- **분할 정복** 처럼 중앙값을 캐치. 나눠진 두 부분의 중앙값을 또 캐치
- **트리의 레벨**을 구분하기 위해 **bfs의 큐 사이즈** 도입
  - 자식노드만큼 큐에 push 되기 떄문.
