# 1학년

- https://www.acmicpc.net/problem/5557

## 해설
map, bfs?, dp?

- bfs로 피연산자 하나하나 + 와 - 해준다.
    - 이때, 중복되는 부분의 개수를 적어주기 위해 qu대신 map을 사용한다.
        - <결과 값, 다향식 개수>
        - ex)
        - (3, 1) => 1 => (4, **2**, 0)
        - 이때 2는 2가지 방법. (3 - 1, 1 + 1).
        - 이런경우를 map에 (2, 2)로 저장.
- 주의
    - long 사용.
    - 중간에 가능한 수는 0 ~ 20의 수이다.
